## Color palette:

(Heretic) https://zdoom.org/wiki/Palette

## Run it:

#### 1. Step
Run:
```
make
```
#### 2. Step
Open index.html in your browser

## Check for TypeScript validity:

TS compilation should run without errors:

```
npx tsc -p tsconfig.json
```
